1. Create textbox
2. Link textbox to cell behind textbox
3. Add filter function to cell in new table referencing search table

>[!Note] Excel Function
> =FILTER(table,ISNUNBER(SEARCH(searchValue,searchRange)),"None") -- "None" Is returned if nothing is found
